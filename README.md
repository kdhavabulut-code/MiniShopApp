# MiniShopApp

MiniShopApp is a simple shop application developed with WPF (.NET).

## Features
- Category and product listing
- Shopping cart functionality
- Add and remove products from cart
- Order confirmation and cancellation
- Total price calculation

## Technologies
- C#
- WPF (.NET)
- XAML

## Project Status
Work in progress.
This project is developed step by step as part of a learning process.
